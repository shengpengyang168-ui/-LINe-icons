## 项目简介
这是一个收集和展示 LINE 图标的项目，包含网页预览功能，方便查看和选择图标资源。

## 文件结构LINe icons/
├─ index.html # 主页面
├─ css/ # 样式文件（可选）
├─ js/ # 脚本文件（可选）
├─ images/ # 图标或相关图片（可选）
└─ README.md # 项目说明文档

## 功能
- 在网页上展示 LINE 图标
- 支持图标预览和选择
- 响应式布局，适配不同设备
- 项目结构清晰，便于后续扩展
